# YBT Digital

A simple PHP-based e-commerce website for selling digital products.

## Prerequisites

- XAMPP (or any other local server environment with Apache, MySQL, and PHP)

## Setup

1.  **Place the project folder** inside your `htdocs` directory.
2.  **Start Apache and MySQL** from the XAMPP Control Panel.
3.  **Database Configuration**:
    - Create a new database in phpMyAdmin.
    - Import the necessary tables (`users`, `products`, etc.).
    - Update the database credentials in `database/db_connect.php` to match your local environment.

4.  **Access the site** by navigating to `http://localhost/YBT%20Digital` in your browser.

## Features

- User registration and login.
- Display of featured products on the homepage.
- Product detail pages.
